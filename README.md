## 开发人员必看

> 为了方便开发小伙伴协作<br>
> 有一些以下的建议

- 开发工具
    + 前端最好使用**VScode编辑器开发代码，使用微信开发者工具查看效果**;
    + 使用 `vscode` 记得帮自己编辑器安装 **小程序插件** -----> `minapp`;
- 提交代码
    + 每人有一个自己 `GitHub` 账号;
    + **一定一定要使用** `git` 来提交代码;
- 遇到 bug 或者 问题一定要提 issues
